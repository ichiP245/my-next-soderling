# my-next-soderling

My Next Soderling es un proyecto que busca crear un modelo de Machine Learning capaz de predecir partidos del ATP masculino de tenis con una precisión capaz de obtener ganancias en los mercados de apuestas.

El objetivo del proyecto está, sin embargo, poder aprender a predecir el mundo real con datos de público acceso. Las eventuales ganancias económicas son solo una consecuencia de haber planteado y desarrollado un buen modelo.

Los datos fueron obtenidos de http://www.tennis-data.co.uk/. Se extrajeron de allí, 275 archivos de formato CSV, cada uno conteniendo todos los partidos de 2015 hasta 2025 de torneos Grand Slam, Master 1000 y ATP 500. La cantidad de partidos (filas) registradas ronda las 16 mil. Se evaluará al final del modelo si vale la pena incorporar los torneos ATP 250, sabiendo que eso aumentará el volumen de datos disponibles, pero no sabemos si esos datos cuentan con el mismo nivel de granularidad y detalle con los que contamos en los torneos mencionados.
